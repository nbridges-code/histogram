# histogram
This program creates a histogram of the word count of a text file. It uses a concurrent buffer that implement a producer-consumer pattern and operates with help from POSIX threads.

```
./histogram wap.txt

6 48559
7 40294
8 28249
9 16466
10 10122
11 4065
12 2573
13 1300
14 408
15 124
16 57
17 9
18 3
19 0
20 0
```
